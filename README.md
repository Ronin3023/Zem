# ZEM Logistics Leave Dashboard

Open `index.html` for the dashboard.
Open `leave-requests.html` for the complete Leave Requests management page.

Files:
- `index.html` — dashboard
- `leave-requests.html` — leave requests page
- `styles.css` — shared styling
- `script.js` — dashboard behavior
- `leave-requests.js` — leave requests behavior
- `assets/` — ZEM logo and admin avatar

The Leave Requests page supports search, status/type filters, pagination, approve/reject actions, adding new requests, and local browser storage.
